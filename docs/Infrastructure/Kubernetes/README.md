# ☸️ Kubernetes

Kubernetes(K8s)는 컨테이너화된 애플리케이션의 배포, 확장 및 관리를 자동화하는 오픈소스 시스템입니다. 이 섹션에서는 쿠버네티스의 일반적인 아키텍처와 핵심 개념, 그리고 CKA 자격증 준비를 위한 상세 가이드를 제공합니다.
