# Architecture (시스템 아키텍처)

소프트웨어 및 시스템 아키텍처 설계에 필요한 핵심 개념과 패턴을 정리한 공간입니다.

## 목차
1.  **[High Availability (고가용성)](High_Availability.md)**: 시스템의 다운타임을 최소화하고 지속적인 운영을 보장하기 위한 전략 (이중화, Failover, HA 클러스터링 등)
2.  **[Pipeline (파이프라인)](Pipeline.md)**: 데이터 처리 효율성과 자동화를 위한 파이프라인 아키텍처 (ETL, CI/CD, Instruction Pipeline)
