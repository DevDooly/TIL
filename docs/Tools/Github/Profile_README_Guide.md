# GitHub 프로필 메인 페이지(Profile README) 꾸미기 가이드

GitHub의 프로필 페이지는 개발자의 첫인상을 결정하는 중요한 공간입니다. 자신만의 개성을 담은 **Profile README**를 만드는 방법과 유용한 도구들을 정리합니다.

---

## 1. 시작하는 방법

1. 자신의 GitHub 아이디와 **동일한 이름의 저장소(Repository)**를 생성합니다.
    * 예: 아이디가 `devdooly`라면 저장소 이름도 `devdooly`.
2. 해당 저장소를 **Public**으로 설정하고, **Add a README file** 옵션을 체크합니다.
3. 저장소 상단에 "✨ Special repository"라는 문구가 뜨면 성공입니다. 이 저장소의 `README.md`를 수정하면 프로필 메인에 노출됩니다.

---

## 2. 주요 구성 요소 및 기술

### 2.1 기술 스택 뱃지 (Shields.io)
가장 많이 쓰이는 도구로, 깔끔한 기술 아이콘과 뱃지를 제공합니다.

* **사용 예시**: `https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white`
* **팁**: Simple Icons(simpleicons.org)에서 서비스별 로고 이름을 확인할 수 있습니다.

### 2.2 GitHub Stats (GitHub Readme Stats)
내 활동 내역(커밋 수, 별점, PR 등)을 카드 형태로 보여줍니다.

* **레퍼런스**: [github-readme-stats](https://github.com/anuraghazra/github-readme-stats)
* **기능**: Top Languages, Stats Card 등 제공.

### 2.3 다이내믹 요소 (GitHub Actions 활용)

* **Blog/RSS 연동**: 최근 포스팅한 블로그 글 자동 노출.
* **WakaTime 연동**: 한 주간의 코딩 시간 통계 노출.
* **Capsule Render**: 프로필 상단에 멋진 웨이브나 그라데이션 타이틀 배너 추가.

---

## 3. 추천 도구 및 제너레이터

직접 마크다운을 작성하기 어렵다면 아래의 제너레이터를 활용해 보세요.

1. **[GitHub Profile README Generator](https://rahulchakraborty.github.io/github-profile-readme-generator/)**: 설문을 채우듯 입력하면 자동으로 마크다운 코드를 생성해 줍니다.
2. **[GPRG (GitHub Profile README Generator)](https://profilereadme.com/)**: 더 현대적이고 다양한 테마를 제공합니다.
3. **[Canva](https://www.canva.com/)**: 프로필 상단 메인 배너를 직접 디자인할 때 유용합니다.

---

## 4. 참고할 만한 멋진 프로필 (Awesome Lists)

수많은 개발자들의 창의적인 프로필을 구경하고 아이디어를 얻어보세요.

* **[Awesome GitHub Profile README](https://github.com/abhisheknaiidu/awesome-github-profile-readme)**: 카테고리별(섹시한, 심플한, 동적인 등) 최고의 프로필 모음집.
* **[GitHub Profile README Gallery](https://github-readme-gallery.netlify.app/)**: 전 세계 개발자들의 실제 프로필 갤러리.

---

## 5. 꾸미기 팁 (Best Practices)

1. **과유불급**: 너무 많은 애니메이션과 이미지는 로딩 속도를 늦추고 시선을 분산시킵니다.
2. **연락처 명시**: 이메일, LinkedIn, 블로그 주소를 뱃지 형태로 눈에 띄게 배치하세요.
3. **목표와 관심사**: 현재 공부 중인 기술이나 참여 중인 프로젝트를 간단히 언급하는 것이 좋습니다.
4. **다크 모드 배려**: 이미지 배경이 투명이 아닌 경우 다크 모드에서 어색해 보일 수 있으므로 PNG(투명 배경) 사용을 권장합니다.

---

## 요약
GitHub 프로필은 당신의 **기술적 이력서**와 같습니다. **Shields.io**로 깔끔한 기술 스택을 보여주고, **GitHub Readme Stats**로 활동성을 증명하며, 본인만의 독특한 개성을 담아 꾸며보세요!
