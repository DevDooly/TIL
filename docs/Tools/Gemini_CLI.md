# Gemini CLI

Gemini CLI는 터미널 환경에서 실행되는 대화형 AI 에이전트입니다. 소프트웨어 엔지니어링 작업에 특화되어 있으며, 코드베이스 분석, 리팩토링, 버그 수정, 문서 작성 등 다양한 작업을 수행할 수 있습니다.

## 주요 기능

*   **코드 이해 및 수정**: 프로젝트의 코드 스타일과 컨벤션을 준수하여 코드를 읽고 수정합니다.
*   **쉘 명령어 실행**: 필요한 경우 안전하게 쉘 명령어를 실행하여 파일 시스템 조작, 테스트 실행 등을 수행합니다.
*   **컨텍스트 인식**: 현재 작업 디렉토리와 프로젝트 구조를 인식하여 문맥에 맞는 답변을 제공합니다.

## 사용 팁

1.  **구체적인 지시**: "버그 고쳐줘"보다는 "로그인 함수에서 발생하는 NullPointerException을 해결해줘"와 같이 구체적으로 요청하는 것이 좋습니다.
2.  **컨텍스트 제공**: 관련된 파일이나 에러 로그를 함께 언급하면 더 정확한 해결책을 찾을 수 있습니다.
3.  **단계적 접근**: 복잡한 작업은 여러 단계로 나누어 요청하면 에이전트가 더 효율적으로 처리할 수 있습니다.
4.  **메모리 활용**: "내 프로젝트의 메인 브랜치는 `main`이 아니라 `develop`이야, 기억해둬"와 같이 프로젝트 정보를 기억시켜 추후 대화에 활용할 수 있습니다.

## 명령어 (Commands)

Gemini CLI는 자연어 대화 외에도 다음과 같은 슬래시 커맨드를 지원합니다.

*   `/help`: 도움말 정보를 표시합니다. 사용 가능한 명령어와 기본적인 사용법을 확인할 수 있습니다.
*   `/bug`: 버그를 신고하거나 피드백을 보낼 때 사용합니다.
